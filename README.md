## 使用方式

>CTRL+/

## 当前支持的文件(vue,wxml)
```html
<template>
    <h2>eg:</h2>
     <div>
        <!-- <div></div>  -->
        <div></div>
        <!-- <div></div>  -->
     </div> 

    <h2>eg:</h2>
       <!-- <div> -->
       <!-- <!~~ <div></div>  ~~> -->
       <!--  <div></div>   -->
       <!-- <!~~ <div></div>  ~~> -->
       <!-- </div>  -->
</template>
```
